## 🚀 About

**SysCut** is a modern web-based video editing platform designed for speed, simplicity and professional workflows.

Unlike traditional desktop editors, SysCut runs directly in your browser while providing powerful timeline editing, animations, effects, keyframes and media management.

## 📦 Installation

```bash
git clone https://github.com/bogajoss/syscut.git

cd syscut

pnpm install
```

---

## 🚀 Development

```bash
pnpm dev
```

Open:

```text
http://localhost:3000
```

## 🔨 Production Build

